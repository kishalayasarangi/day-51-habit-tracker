# Day 51 — Habit Tracker

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A habit tracker with streak tracking, heatmap calendar
and daily check-in system.

## How to view
Open `index.html` in your browser.

## Features
- Add and track daily habits
- Streak counter per habit
- GitHub-style heatmap calendar
- Weekly and monthly stats
- localStorage persistence
